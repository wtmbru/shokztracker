# Shokz VPL Daily Performance Tracker

A single-file web app for tracking daily Shokz product sales, GSP attachments, and hourly revenue — built for VPL reps.

## Features

- **Hourly revenue tracking** across six time blocks with an auto-calculated daily total
- **Product tally matrix** for all 11 Shokz models with +/− stepper buttons for Units Sold and GSP Attached
- **Shift totals** that update in real time (total units, total GSP, total revenue)
- **Export** — generates a formatted day summary you can copy and forward directly to your manager
- **Reset** — clears all fields for a fresh shift

## Usage

No install, no dependencies. Just open `tracker.html` in any browser.

1. Set your date and shift time at the top
2. Enter revenue for each hour block as the shift progresses
3. Use the **+** / **−** buttons to log units sold and GSP attachments per product
4. Totals calculate automatically
5. Hit **Export Day Summary** to generate a text report, then copy and send it
